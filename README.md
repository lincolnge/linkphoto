# PhotoWall

## Introduce

该项目可以记录每天运动量。采用 fullcalendar + Django 搭建，数据库为 MySQL

## Create Database

CREATE DATABASE app_linkphoto DEFAULT CHARACTER SET gbk COLLATE gbk_chinese_ci;

## Bower

In Dir photowall/static/

	$ bower install jquery-ui
	$ bower install fullcalendar

## Gulp

	$ gulp
	$ gulp serve
